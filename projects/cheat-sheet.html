JavaScript Object Basics

In JavaScript, variables hold data values. Objects are also variables, but can contain many data values instead of just one. Here's a basic object: 


		var ian = {
			name: "Ian Lockwood",
			birthday: "March 30, 1993",
			age: 22
		}

As you can see, we can create a new object just as we would a variable, with the var keyword. We separate elements with commas, as you might with a ruby array or hash.

name: "Ian Lockwood" is a name:value pair known as a property. Properties are the most fundamental parts of objects, and we'll go into detail on them in the next section.

JavaScript Object Properties

JavaScript objects are made up of properties, the key:value pairs that describe an object.

Adding Properties

Adding properties is as easy as writing them into the object in the syntax seen in the basics section. However, we can add properties without editing the original code by using the following syntax:


		objectName.newProperty = value

Deleting Properties

To delete properties, use the following syntax: 


		delete objectName.property

Accessing Properties

To access properties, use any of the following syntaxes: 


		objectName.property

		objectName["String Key"]

		objectName[expression_or_variable]

Looping Through Properties

To loop through an object's properties, you can use the handy for...in loop: 


		for (property in objectName) {
    			code block
		}

JavaScript Object Methods

JavaScript object methods are one of the handiest components of JavaScript Objects. An object method is a property that contains a function.


		var ian = {
		method : function() { code block},
		}

Pretty simple! And accessing and using objects is simple too. Just use the following syntax:


		objectName.method()

Don't forget to use those parentheses! If you do, JavaScript will return the function definition.
